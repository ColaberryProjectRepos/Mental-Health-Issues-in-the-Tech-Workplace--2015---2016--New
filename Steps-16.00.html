Adding Date Slicer 16

1.   In the report view of the dashboard, open the visualization pane, then Select the Slicer.
2.   In the fields pane, open Calendar Table, drag the Year.
2a. In the Formatting Tab-->Visual-->Slicer Settings: Between; Slicer Header = Off; Values: Font = Segoe UI = 9, Color = black, No Background; Slider: On, Color = black
2b. In the Formatting Tab-->General-->Properties-->Height = 97, Width = 222, Position: Horizontal = 683, Vertical = 0, Title: Year, Font = DIN, Size = 12, Background: White, Effect: Background: On, Color = White, Visual Border = Off; Color = White; Shadow = Off.
