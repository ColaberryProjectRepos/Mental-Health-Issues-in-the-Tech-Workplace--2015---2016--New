Creating Job Title Scatter Chart 21

  • Click on an empty space on the canvas, and from the visualizations pane select Scatter Chart.
    •  Fill up figures provided below for the necessary fields.
    • Build Visual Tab:
    • Values = Job Title
    •  X-axis = Total Cases
    •  Y-axis = Cases YoY%
    •  Legend = Were you aware of the options for mental health care provided by your previous employers.
    • Size = Count of How many employees does your company or organization have
    •   Tooltips = TotalCases, Awareness%, Benefits%, Discussions%, TakeSerious%
    • Tooltip = Keep all filters on.
Format Visual Tab:
Visual:
  • X-axis: Range: Minimum=Auto, Maximum=50; Logarithmic scale, On; Invert range, Off; Values: Font=Segoe UI (9), Font color=#605E5C; Display Units=Auto; Value Decimal places=Auto; Show blank values = Off; Title = On; Title text = Auto, Style = Show title only, Font = DIN (12), Color = #E66C37.
    • Y-axis: Range: Minimum= -2, Maximum=Auto; Logarithmic scale, Off; Invert range, Off; Values = On; Font=Segoe UI (9), Font color=#605E5C; Display Units=Auto; Value Decimal places=Auto; Switch Axis Position: Off; Show blank values, Off; Title = On; Title text = Auto, Style = Show title only, Font = DIN (12), Color = #E66C37. 
  • Legend=On, 
    • Position=Top left; Text: Segoe UI (8) Bold, Color=#605E5C; Title=Off.
    • Gridlines: Horizontal=On; Style = Solid, Color=#F3F2F1,Width=1 px; Vertical: On, Style: Dotted, Color = #C8C6C4, Width, 1 px; Zoom slider=Off.     
      • Markers: Apply Setting to|Series: All; Shape: Type, circle; Color Border, On; Range scaling = Auto, Size (16); Color: I was aware of some = #EC8FCA; Not currently aware = #F3B4DC; No, I only became aware = #F3B4DC; Yes, I was aware of all of the = #E044A7. 
    •  Category Label: On; Values: Font, Segoe UI (9), Color = #605E5C; Background = Off.
  • Plot Area Background: NA.
