Creating Mental Health YoY% Waterfall Chart 19

1.  To create this chart we need to first create a measures table.


To do this, On Power BI Desktop Home ribbon, Click on Enter data, name first column header as ID and second column header measures.  
 2. Name the table Waterfall and create your Variance measure within the table.

Variance = SWITCH(
  SELECTEDVALUE('Waterfall'[ID]),
    1, [Cases YoY%],
    2, [Aware YoY%],
    3, [Ben YoY%],
    4, [Disc YoY%],
    5, [Ser YoY%])

3. Click on an empty space on the canvas, and from the visualizations pane select Waterfall Chart.
4. Drag the Measures column from Waterfall table to the Category.  Then drag the Variance measure into Y-axis, Drag Cases YoY%, Aware YoY%, Ben YoY%, Disc YoY%, and Ser YoY%  into Tooltips field.
5. In the Formatting Tab-->Visual -->X-axis: On, Font = Segoe UI, size = 9, Color = #605E5C. Title: Off. 
Y-axis: Range: Minimum = Auto, Maximum = Auto, Invert range, Off; Values: On, Font = Segoe UI, size = 9, Color = #605E5C. Title: Off.  
6. Legend: On, Position = Top left, Text: Font = Segoe UI, (10), Color = #605E5C. Title: Off.
Gridlines: Horizontal: On, Style = Solid, Color = #F3F2F1, width 1px.
7. Columns: Color Increase = #D64550, Decrease = #298A60, Total = #A38600.
8. Data labels: On. Options|Orientation = Horizontal, Position = Outside end; Values: Font = Segoe UI, (10), Color = #605E5C, Background: Off.
9. In the Formatting Tab-->General-->Properties-->Size: Height = 321, Width = 378, Position: Horizontal = 553, Vertical = 96; Advanced Options|Responsive: On, Maintain Layer Order = Off.
10. Title: Mental Health YoY%, Font = DIN Bold, Size = 14, Text Color = #252423, Background: #EC8FCA, Center; Subtitle: Off; Divider: On, Color = #605E5C, Style = Solid, Width 2px, Ignore padding, on; Spacing|Customize Spacing: Off; Vertical between label, 4 px.
